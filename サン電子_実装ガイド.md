# サン電子株式会社 基板進捗管理システム
## 実装ガイド

---

## システム概要

遊技機の機種ごと・基板ごとに「新規設計フロー」または「既存流用フロー」を管理するシステムです。
設計課・資材購買課・営業課・品質管理課の4部署が、それぞれの担当ステップで「ボールを渡す」操作を行います。

---

## 業務フロー詳細

### 新規設計フロー（7フェーズ）

```
[設計課] 仕様検討 → 回路設計 → 試作・検証
         ↓
[資材購買課] BOM・部材確定 → 量産製造発注
         ↓
[品質管理課] 受入検査
         ↓
[営業課] お客様納品
```

**各フェーズのSLA（目安）**:
- 仕様検討: 10営業日（240h）
- 回路設計: 20営業日（480h）
- 試作・検証: 20営業日（480h）※品質管理課の初回品確認含む
- BOM・部材確定: 5営業日（120h）
- 量産製造発注: 2営業日（48h）
- 受入検査: 2営業日（48h）
- お客様納品: 1営業日（24h）

### 既存流用フロー（4フェーズ）

```
[営業課] 受注処理（見積〜注文書受領）
         ↓
[資材購買課] 製造発注（外部工場）
         ↓
[品質管理課] 受入検査
         ↓
[営業課] お客様納品
```

## 各部署の操作方法

### 共通：ボールを渡す操作

1. ダッシュボードを開く（URL）
2. 「部署ビュー」タブ → 自分の部署を選択
3. 担当カードをクリック → 詳細モーダルが開く
4. 「実施者名」に自分の名前を入力
5. 「コメント・引継ぎメモ」に次担当者へのメモを記入
6. 「▶ 次のステップへ」をクリック → 次の部署へ自動通知

### 品質管理課：QC判定の登録

詳細モーダルに「ロット番号」と「QC判定」欄が表示されます:
- **合格**: 次フェーズへ自動移行
- **不合格**: 「NG差し戻し」ステータスになり、設計課または資材購買課に通知
- **保留**: 追加確認が必要な場合

### 営業課：見積管理システムとの連携

注文書番号を入力して基板フローを登録すると、毎朝8時に見積管理GASから:
- 納品予定日を自動取得・更新
- 注文書の変更があればSLA期限も自動調整

## 機種販売表の登録手順

1. お客様から機種販売表（Excel/PDF）を受け取る
2. ダッシュボード右上「＋ 基板登録」をクリック
3. 機種コード・機種名を入力
4. 搭載される各基板（M基板、D基板等）ごとに登録
5. 各基板を「新規設計」か「既存流用」かを選択してフローを開始

**ポイント**: 1機種に複数の基板がある場合は、基板ごとに別々のフローとして登録します。
例: A77機種 → M基板（新規設計）+ D基板（既存流用）+ E基板（既存流用）で3件登録

## 画面説明

### 機種ビュー
- 機種コードごとに折りたたみ表示
- 各機種の基板が一覧化され、全体の進捗を把握
- ヘッダーの●（色）でその機種の最悪SLA状態を一目確認

### 部署ビュー
- 自部署のボール（担当中案件）だけを表示
- カードの左側ライン: オレンジ=SLA警告, 赤=危険/超過
- 「設計課」「資材購買課」「営業課」「品質管理課」タブで絞り込み

### 基板一覧
- 全基板をテーブルで確認
- フロー種別・基板種別・ステータスでフィルタリング
- 納品予定日の残り日数表示（負数=超過）

## 既存の見積管理システムとの関係

```
mitumorikannri-VR3（既存）
  └─ 見積書・注文書管理
  └─ 注文書の納期 → 毎朝8時に自動同期
                     ↓
本システム（新規）
  └─ 注文書番号でリンク
  └─ 納品予定日を取得してSLA期限に設定
  └─ 基板ごとの部署間ワークフローを管理
```

## よくあるQ&A

**Q: 1機種に同じ種別の基板が複数ある（M基板-1、M基板-2など）**
A: 基板IDを「M2601A」「M2601B」のように別IDにして、それぞれ個別に登録してください。

**Q: 設計途中でスペック変更が発生した**
A: 詳細モーダルの「保留」ボタンで一時停止し、メモに変更内容を記録。再開時に「次のステップへ」で進行再開。

**Q: 外部工場の製造が遅れている**
A: 資材購買課の「製造進捗確認」ステップで滞留します。設定時間（デフォルト72時間）を超えると自動アラートが発送されます。

**Q: QC不合格の基板はどうなる**
A: ステータスが「NG差し戻し」になります。管理者がNG内容を確認し、設計課（新規設計フロー）または資材購買課（既存流用フロー）に手動でステータスを「進行中」に戻して再フローを開始してください。
